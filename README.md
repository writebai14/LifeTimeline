
# LifeTimeline · 个人时间轴日志

**LifeTimeline**  是一个本地优先（Local-First）的人生时间轴记录工具。

它可以帮助你用较小的时间颗粒度记录每天做过的事情，逐渐构建出一条属于自己的人生时间轨迹。
与传统的效率工具不同，LifeTimeline 更关注 **记录生活与自我反思**，而不是任务管理。

---

## 项目理念

> 子曰：“逝者如斯夫，不舍昼夜。”
  
时间像河流一样向前流去，从不为任何人停留。
  
我们每天都在生活，却很少真正留下关于时间的记录。
当回望过去时，我们往往只能记住一些模糊的片段。

十年前的今天，此刻的你在做什么？
当时在关心什么事情？
在思考什么问题？
又和今天的自己有什么不同？

那些曾经重要的人和事，是否已经渐渐模糊？

**LifeTimeline 希望做一件简单的事情：**

记录生活中的时间片段，留下属于自己的时间轨迹。
当很多年以后再次回看，你仍然可以回到那个时刻，看见当时的自己。

---

## 功能特性

- ⏰ **时间块记录**  
  以细粒度时间块记录每日活动，构建完整的生活时间轴。
- 🧠 **直观的时间轴视图**  
  简洁清晰地回顾一天、一周或更长时间的生活轨迹。
- 📷 **媒体记录支持**  
  支持添加照片、截图等媒体内容，让回忆更加具体和真实。
- 🔒 **本地优先，隐私至上**  
  所有数据默认存储在本地，用户拥有 **100% 的数据所有权**。
- ⚡ **轻量快速，无复杂配置**  
  打开即可使用，不依赖复杂环境或云服务。
- 🔄 **纯文本互通，数据零束缚**  
  支持与纯文本记录方式（如 Apple 备忘录等）进行 **导入 / 导出**，  
  让数据始终掌握在用户手中，不被任何系统锁定。

## 快速开始

### 1 克隆项目

```bash
git clone https://github.com/writebai14/LifeTimeline.git
cd LifeTimeline
```
``
### 2 安装依赖

1. **安装 Node.js**  
   若尚未安装，请到 [nodejs.org](https://nodejs.org/) 下载并安装 LTS 版本。
2. **在终端中进入项目目录**  
   打开「终端」或「命令提示符」，执行：
   ```bash
   cd /Users/LifeTimeline
   ```
3. **安装依赖（只需做一次）**  
   ```bash
   npm install
   ```

### 3 运行项目
   ```bash
npm run dev
   ```

然后在浏览器中打开：
   ```bash
http://localhost:5173
   ```

终端里会提示本地地址，一般为：  
 http://localhost:5173
用浏览器打开该地址即可使用。

### 4 停止服务

关闭终端或按 `Ctrl+C` 可停止服务；下次使用再次执行 `npm run dev` 即可。

## 使用说明

- **切换日期**：顶部通过「← →」或点击日期切换查看/编辑某一天。
- **一句话**：点击「一句话」，输入当前在做的事，会按当前时间生成一个时间块，可稍后再编辑。
- **拍照/截图**：点击「拍照/截图」选择本地图片，会作为「待补记素材」出现在当天页面底部，可「生成时间块」或「关联到已有块」。
- **添加时间块**：点击「+ 添加时间块」或点击已有块可编辑起止时间、描述、分类、地点、备注等。
- **数据位置**：所有数据保存在项目下的 `data` 文件夹内（按日期 JSON + 媒体文件），可整体复制备份。

---

## 数据存储

LifeTimeline 采用 **本地优先（Local-First）** 的设计。

所有用户数据都存储在本地目录：
```bash
data/
 ├─ days/
 └─ media/
```
